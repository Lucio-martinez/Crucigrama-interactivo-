# Crucigrama-interactivo-
Esta actividad promueve la participación, la reflexión y la autoevaluación, ya que ofrece retroalimentación inmediata sobre las respuestas ingresadas. Además, estimula habilidades cognitivas como la atención, la memoria comprensiva y la asociación conceptual, fortaleciendo el dominio del vocabulario académico en un entorno dinámico e interactivo.
